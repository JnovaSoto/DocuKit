    âš¡ Installation & Setup

    1. Clone & Install:

      git clone https://github.com/JnovaSoto/DocuKit.git
      cd DocuKit
      npm install

    2. Environment Configuration: Create a .env file in the root:

      PORT=3000
SESSION_SECRET=your_super_secret_key
NODE_ENV=development
GOOGLE_CLIENT_ID=your_google_id
GOOGLE_CLIENT_SECRET=your_google_secret
DATABASE_URL="file:./db/database.sqlite"

    3. Initialize Database:

      # This synchronizes the schema and seeds initial data via Prisma
      npm run create-db

    4. Build & Develop:

      # Compile TypeScript
      npm run build

      # Run in development mode (auto-reload)
      npm run dev

    5. Database Exploration:

      # Open Prisma Studio to view and edit your data
      npm run studio
